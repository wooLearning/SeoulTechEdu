Presentation-oriented summary of the current address space for RISC-V_Multi_Cycle. The address decode follows Top_Memory_CNTL.sv and Top_APB.sv, while register names and software-visible meanings follow test_source/mmio.h.

Access behavior

Misaligned or unsupported access sizes are rejected in Top_Memory_CNTL.sv. Addresses outside ROM, RAM, and MMIO also return an error response instead of silently aliasing.

Peripheral Base Address Map

All active APB peripherals are spaced on 4 KB boundaries for clean decode and simple firmware constants.

GPO

0x2000_0000

16-bit general-purpose output page. Firmware controls output-enable mask and output data value.

GPI

0x2000_1000

16-bit general-purpose input page. Used by the demo to sample switches and control UART/FND behavior.

GPIO

0x2000_2000

Bidirectional GPIO page. Separate direction register, output register, and observed input register.

FND

0x2000_3000

Seven-segment counter page. Minimal one-register interface for run and stop control.

UART

0x2000_4000

UART TX/RX page with FIFO status, sticky error clear bits, and baud-source selection register.

Register Map by Peripheral

Grouped layout for slide capture: one block per peripheral instead of one long flat table.

GPO Register Page

Base = 0x2000_0000

Register Address Access Meaning
GPO_CTL 0x2000_0000 R/W Per-bit output-enable mask. Bit = 1 drives the corresponding GPO pin.
GPO_ODATA 0x2000_0004 R/W Output value applied to bits enabled in GPO_CTL.

GPI Register Page

Base = 0x2000_1000

Register Address Access Meaning
GPI_CTL 0x2000_1000 R/W Enable mask for the sampled GPI bits exposed back to software.
GPI_IDATA 0x2000_1004 R Masked input sample. Demo firmware interprets bits for baud select, source select, FND run, and GPIO pattern.

GPIO Register Page

Base = 0x2000_2000

Register Address Access Meaning
GPIO_CTL 0x2000_2000 R/W Direction control. Bit = 1 means output, bit = 0 means input.
GPIO_ODATA 0x2000_2004 R/W Output value driven when the matching direction bit is configured as output.
GPIO_IDATA 0x2000_2008 R Observed input value for the GPIO bus.

FND Register Page

Base = 0x2000_3000

Register Address Access Meaning
FND_RUN 0x2000_3000 R/W Bit0 controls the counter. 0 = stop, 1 = run.

UART Register Page

Base = 0x2000_4000

Register Address Access Meaning
UART_ID 0x2000_4000 R Peripheral identifier. Expected value is 0x5541_5254.
UART_STATUS 0x2000_4004 R TX/RX FIFO state, TX busy, RX overflow, and frame error flags.
UART_TXDATA 0x2000_4008 W Push one byte to TX FIFO when space is available. Write while full returns slave error.
UART_RXDATA 0x2000_400C R Pop one byte from RX FIFO. Read while empty returns slave error.
UART_CONTROL 0x2000_4010 R/W Sticky error clear control and status mirror for overflow and frame-error bits.
UART_BAUDCFG 0x2000_4014 R/W Selects switch-vs-APB baud source and records the currently active baud select code.

Demo Firmware Defaults and Presentation Notes

Default software setup from mmio.h

GPO_CTL = 0xFFFF, GPO_ODATA = 0x0000, GPI_CTL = 0x07FF, GPIO_CTL = 0x000F, GPIO_ODATA = 0x0005, FND_RUN = 1, and the UART defaults to switch-selected 115200.
