# 문서 폴더 안내

`md/` 폴더는 `RISC-V_multicycle` 프로젝트의 보고서, 시각 자료, 데이터 표, build report를 모아둔 공간입니다.

## 폴더 구성

- `reports/`: Markdown 보고서와 실행 로그
- `data/`: CSV, HTML 형식의 보조 자료
- `notebooks/`: UART 검증 시각화에 사용한 노트북
- `visuals/figures/`: 차트, heatmap, PNG/SVG 그림
- `visuals/diagrams/`: 시스템 다이어그램
- `build_reports/`: timing, utilization, power report
- `reference/`: `mmio.h` 같은 참조 자료
